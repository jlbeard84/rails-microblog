# rails-microblog

[![Build Status](https://travis-ci.org/jlbeard84/rails-microblog.svg?branch=master)](https://travis-ci.org/jlbeard84/rails-microblog)

Simple rails microblog app.

## Getting Started

    $ bundle install --without production
    
## Test

    $ rails test
    
## Serve

    $ rails serve
